# Actions-OpenWrt

使用[P3TERX/Actions-OpenWrt](https://github.com/P3TERX/Actions-OpenWrt)的模板对lean's lede源码进行编译

## 超纯净版X86_64 OpenWrt
已包含插件wireguard、ttyd终端、openclash、全能推送、dns-go、upnp、openvpn服务器、带宽监控
